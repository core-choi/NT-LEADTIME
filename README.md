# NT 대시보드 자동 업로드 시스템

## 폴더 구조

```
바탕화면 (또는 원하는 위치)/
├── 업로드.bat              ← 더블클릭하면 업로드 실행
├── 매일자동등록.bat         ← 매일 9시 자동 실행 등록 (1회만)
├── 엑셀넣기/               ← 여기에 엑셀 파일만 넣으면 끝!
│   ├── 현월/
│   │   ├── 서비스_리드타임_현황.xlsx
│   │   └── RoReport.xlsx
│   ├── 전월/
│   │   ├── 서비스_리드타임_현황.xlsx
│   │   └── RoReport.xlsx
│   └── 전년/
│       ├── 01/ ~ 12/
└── NT-LEADTIME/            ← 자동 생성됨 (건드리지 않기)
```

## 사용법

### 방법 1: 수동 업로드 (더블클릭)
1. `엑셀넣기/현월/` 폴더에 엑셀 파일 2개 넣기
2. `업로드.bat` 더블클릭
3. 1~2분 후 웹페이지 자동 갱신

### 방법 2: 매일 자동 업로드
1. `매일자동등록.bat` 우클릭 > 관리자 권한으로 실행 (1회만)
2. 매일 오전 9시에 자동 실행
3. 엑셀 파일이 있으면 업로드, 없으면 아무 일도 안 함

### 매월 교체
1. 현월 엑셀 → `엑셀넣기/전월/`로 이동
2. 새 엑셀 → `엑셀넣기/현월/`에 넣기
3. `업로드.bat` 더블클릭 (또는 자동 실행 대기)
